# 读卡器申请流程

## 申请条件：

1.确定电脑已经完成泰坦安装包

2.确定电脑安装适配chrome浏览器

### 申请需满足以下其中一个条件：

1、没有读卡器 

2、老读卡器无法兼容泰坦身份证的驱动 

3、原读卡器用于与公安\旅业上传，连接的电脑是专网专用无法登入泰坦系统。
